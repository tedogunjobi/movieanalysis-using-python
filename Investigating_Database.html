Data Cleaning & Wrangling (Removing the unused information from the dataset )¶

Important observation regarding this process

  1. We need to remove unused column such as id, imdb_id, vote_count, production_company, keywords, homepage etc.
  2. We also need to remove duplicate rows(if any).
  3. Some movies in the database have zero budget or zero revenue as such we need to discard them
  4. Replacing zero with NAN in runtime column.
  5. We need to change the release within the date column into a date format.
  6. Changing format of budget and revenue column.
